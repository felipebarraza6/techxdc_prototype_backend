👥 Gestión de Usuarios

GET /api/users

Obtener todos los usuarios

Ejemplo: GET http://localhost:3000/api/users
GET /api/users/:id

Obtener usuario por ID

Ejemplo: GET http://localhost:3000/api/users/1
POST /api/users

Crear nuevo usuario

Body: 
{
  "username": "usuario1",
  "email": "usuario@ejemplo.com",
  "first_name": "Juan",
  "last_name": "Pérez",
  "password": "password123",
  "rol": "CLIENT",
  "group_id": 1
}
PUT /api/users/:id

Actualizar usuario

DELETE /api/users/:id

Eliminar usuario

GET /summary

Resumen del proyecto (HTML)

Descripción: Muestra una página HTML con el resumen de cambios, problemas encontrados y soluciones aplicadas al proyecto.

🔧 Información Técnica

Base URL:

http://localhost:3000 (o el puerto configurado)

Headers Recomendados:

Content-Type: application/json
Accept: application/json

Respuestas Estándar:

{
  "success": true/false,
  "message": "Descripción del resultado",
  "data": {...},
  "error": "Descripción del error (si aplica)"
}
